想测试一下 `git branch -f` 的用法


过程

1. 我在 `git-branch--f` 分支 试了一下 `git branch -f git-branch--f-branched`，发现没变化，但是查看分支会发现多了一个分支。
2. 上网搜索了一下，发现需要两个参数，`git branch -f [branch_name] [object_pos]`，解释是 **移动分支到目标位置**。
